## 📋 完整操作步驟（5-10 分鐘）

### **第 1 步：訪問 Google Search Console**

1. 打開瀏覽器，訪問：
   ```
   https://search.google.com/search-console
   ```

2. 使用你的 Google 帳戶登入（與 GA4 相同的帳戶）

---

### **第 2 步：添加網站資源**

1. 點擊左上角的 **"添加資源"** 或 **"Add property"**

2. 你會看到兩個選項：
   - **網域（Domain）** - 需要 DNS 驗證
   - **網址前綴（URL prefix）** - 使用 HTML 標籤驗證 ✅ **選這個！**

3. 選擇 **"網址前綴"**，輸入：
   ```
   https://xhsvideodownloader.com
   ```

4. 點擊 **"繼續"** 或 **"Continue"**

---

### **第 3 步：驗證所有權（最簡單的方式）**

1. 在驗證方法中，選擇 **"HTML 標記"** 或 **"HTML tag"**

2. 你會看到類似這樣的代碼：
   ```html
   <meta name="google-site-verification" content="b74c2e3e158d7f83" />
   ```

3. **不需要複製！** 這個代碼已經在你的網站上了！

4. 直接點擊 **"驗證"** 或 **"Verify"** 按鈕

5. 如果成功，你會看到 ✅ **"所有權已驗證"** 的訊息

6. 點擊 **"前往資源"** 或 **"Go to property"**

---

### **第 4 步：提交 Sitemap**

1. 在左側菜單中，找到並點擊 **"Sitemap"** 或 **"站點地圖"**

2. 在 **"新增 Sitemap"** 或 **"Add a new sitemap"** 輸入框中，輸入：
   ```
   sitemap.xml
   ```
   （不需要完整 URL，只需要 `sitemap.xml`）

3. 點擊 **"提交"** 或 **"Submit"**

4. 等待幾秒鐘，狀態會變成：
   - **"成功"** 或 **"Success"** ✅
   - 顯示 **"已發現 5 個網址"** 或 **"5 URLs discovered"**

---

### **第 5 步：請求索引（加速收錄）**

這一步會讓 Google 更快地抓取你的網站。

#### **索引主頁：**

1. 在左側菜單頂部，找到 **"網址檢查"** 或 **"URL Inspection"** 工具

2. 在搜索框中輸入：
   ```
   https://xhsvideodownloader.com
   ```

3. 按 Enter，等待檢查完成（10-30 秒）

4. 如果顯示 **"網址不在 Google 服務中"**：
   - 點擊 **"請求建立索引"** 或 **"Request indexing"**
   - 等待 1-2 分鐘完成

5. 如果顯示 **"網址已在 Google 服務中"**：
   - 太好了！已經被索引了 ✅

#### **索引博客頁面：**

重複上面的步驟，分別檢查並請求索引：

```
https://xhsvideodownloader.com/blog
```

```
https://xhsvideodownloader.com/blog/how-to-download-xiaohongshu-videos
```

## 📊 Search Console 重要功能

### **1. 成效報表（Performance）**
- 查看：點擊次數、曝光次數、點擊率、平均排名
- 了解：哪些關鍵詞帶來流量
- 優化：根據數據調整內容

### **2. 網址檢查（URL Inspection）**
- 檢查：任何頁面的索引狀態
- 測試：頁面是否可以被 Google 抓取
- 請求：加速索引新頁面或更新

### **3. 覆蓋率（Coverage）**
- 查看：哪些頁面已被索引
- 發現：索引錯誤或警告
- 修復：阻止索引的問題

### **4. Sitemap**
- 監控：Sitemap 提交狀態
- 查看：已發現和已索引的 URL 數量
- 更新：提交新的 Sitemap

## ⚠️ 常見問題

### **Q: 驗證失敗怎麼辦？**
A:
1. 確認你訪問的是 `https://xhsvideodownloader.com`（不是 http）
2. 清除瀏覽器緩存後重試
3. 等待 5-10 分鐘後再次點擊 "驗證"
4. 如果還是失敗，告訴我錯誤訊息

### **Q: Sitemap 提交後顯示錯誤？**
A:
- 等待 10-30 分鐘，Google 需要時間處理
- 刷新頁面查看最新狀態
- 如果持續錯誤，告訴我具體錯誤訊息

### **Q: 請求索引後多久生效？**
A:
- 請求提交：立即完成
- Google 抓取：1-3 天
- 出現在搜索結果：1-2 週
- 排名提升：1-3 個月

### **Q: 為什麼我搜索不到我的網站？**
A:
- 新網站需要 1-2 週才會被收錄
- 使用 `site:xhsvideodownloader.com` 搜索查看是否已被索引
- 確保已提交 Sitemap 並請求索引
- 耐心等待，SEO 需要時間
