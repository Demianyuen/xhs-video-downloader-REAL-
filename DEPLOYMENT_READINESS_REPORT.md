## 🏗️ 架构概览

### 技术栈
- **前端框架**: Next.js 16.1.6 (Turbopack)
- **语言**: TypeScript + React 19
- **样式**: Tailwind CSS v3
- **支付**: Stripe
- **广告**: Google AdSense
- **部署**: Vercel

### 核心模块
1. **XHS Service (Vercel 兼容版本)**
   - 使用外部 API 而非系统依赖
   - 支持视频下载、水印移除、转录提取
   - 完整的错误处理和日志记录

2. **AdSense 监测系统**
   - 实时收入监测
   - 自动告警系统
   - 收入预测功能
   - 账户信息管理

3. **API 端点**
   - `/api/download` - 视频下载处理
   - `/api/adsense` - AdSense 监测和报告
   - `/api/transcript` - 转录提取
   - `/api/download/[token]` - 下载令牌处理

4. **前端页面**
   - 主页 (/) - 下载界面
   - 关于页面 (/about) - 项目信息
   - 隐私政策 (/privacy)
   - 服务条款 (/terms)
